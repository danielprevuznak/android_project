Nazov projektu: AmbuBolt

Cielom projektu je vytvorit aplikaciu na sledovanie poloh sanitky

Aplikacia rozlisuje sanitky podla stringu zadaneho pri logine, je potrebne taktiez zadat PINkod

Na sledovanie polohy sa vyuziva Google maps, ktory sa accessuje pomocou API kluca

Dalsia funkcionalita je pridavanie a manazovanie pacientov

Aplikacia taktiez komunikuje s backend serverom pomocou REST API volani a SignalR (realtime)
